Qt（C++）和pyQt的安装记录

Qt（C plus plus）

Qt（Python）

参考此篇文章

  1. 下载PyQt5
pip install PyQt5

网络不好的可选择豆瓣的镜像：

pip install PyQt5 -i https://pypi.douban.com/simple
  1. 下载PyQt5-tools
pip install PyQt5-tools

网络不好的可选择豆瓣的镜像：

pip install PyQt5-tools -i https://pypi.douban.com/simple

PyQt 5.15.x 之前，安装完成后在 %Python_path%\Lib\site-packages 目录中可以看到PyQt5、pyqt5-tools、PyQt5-sip 目录。

PyQt 5.15.x 之后，安装完成后在 %Python_path%\Lib\site-packages 目录中可以看到PyQt5、pyqt5-tools、PyQt5-sip 以及多了的 qt_applications 、qt5-tools 目录。

  1. 添加环境变量

在系统变量Path中添加 pyqt5-tools 目录的绝对路径。

  1. 验证

运行下面代码，出现窗口即为成功。

import sys
from PyQt5.QtWidgets import QWidget, QApplication

app = QApplication(sys.argv)
widget = QWidget()
widget.resize(640, 480)
widget.setWindowTitle("Hello, PyQt5!")
widget.show()
sys.exit(app.exec())
